# NVIDIA-Stock-Analysis
This project analyses historical NVIDIA (NVDA) stock data to understand price trends, volatility, and trading behavior. 

The project includes exploratory data analysis (EDA), financial metrics, and a simple trading strategy based on technical indicators.

The goal is to simulate how a data analyst or financial analyst would evaluate a stock.

## Objectives
- Analyse NVIDIA stock price trends over time  
- Measure volatility and daily returns  
- Identify patterns using technical indicators  
- Build and evaluate a simple trading strategy

## Dataset
- Source: Kaggle  
- Data: Historical NVIDIA stock prices  
- Features include:
  - Date  
  - Open  
  - High  
  - Low  
  - Close  
  - Volume
 
## Methods
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Calculation of daily returns and volatility  
- Moving averages (MA20, MA50)  
- Trading strategy backtesting
